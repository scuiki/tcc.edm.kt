## Resumo de uma linha
Revisão sistemática PRISMA de 198 estudos de EDM publicados entre 2013 e 2023, cobrindo técnicas de ML/DL, estilos de aprendizagem e ética de dados educacionais.

## Fatos críticos para este projeto
- Processo EDM definido em **4 fases**: Problem Definition → Data Preparation → Modelling & Evaluation → Deployment — alinha diretamente com a metodologia deste TCC
- EDM usa: Classificação, Regressão, Clustering, Mineração de Regras de Associação, Detecção de Outliers, Redes Neurais, etc.
- 198 estudos de alta qualidade incluídos (de 2.195 identificados) — revisão PRISMA-compliant
- Cobertura: 2013–2023, publicações em inglês, peer-reviewed

## Metodologia relevante
- 4 fases do processo EDM (referência [10] do paper): Problem definition, Data preparation and gathering, Modelling and evaluating, Deployment
- Técnicas dominantes: Decision Trees, Neural Networks, Naive Bayes, K-NN — além de DL (LSTM, CNN, Transformer)
- Tendências emergentes: Explainable AI (XAI), Blockchain, Multimodal Learning Analytics, Chatbots

## Relevância para este TCC
Útil como referência para a **justificativa metodológica** das 4 fases do pipeline (seção de introdução/metodologia dos notebooks). Citar para embasar a escolha do processo EDM usado neste trabalho.
