## Resumo de uma linha
Estudo comparativo de algoritmos de ML clássicos (RF, LR, SVM, Naive Bayes, KNN) para prever notas finais de 1.854 alunos de um curso universitário turco, alcançando 70–75% de acurácia.

## Fatos críticos para este projeto
- **Tarefa**: performance prediction (prever nota final) — diferente de KT (prever próxima tentativa)
- Dataset: 1.854 alunos, curso Turkish Language-I, universidade turca, semestre 2019–2020
- Features: notas de midterm, dados de departamento e faculdade (apenas 3 tipos de parâmetros)
- Resultado: Random Forest e LR com melhor performance (70–75% acurácia)
- Não usa dados de processo (submissões passo-a-passo) — usa apenas grades agregadas

## Relevância para este TCC
Background para distinguir **Performance Prediction** (prever nota final) de **Knowledge Tracing** (prever próxima tentativa). Útil para justificar a escolha de KT como tarefa principal no CSEDM (dataset de processo, não de grades).
