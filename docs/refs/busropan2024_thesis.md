## Resumo de uma linha
Tese de mestrado (TU Delft) sobre uso de LLMs para learning analytics e intervenções em programação educacional, com foco em personalização de feedback e suporte a estudantes.

## Fatos críticos para este projeto
- Orientador: Prof. Dr. M.M. Specht; Co-orientador: MSc. Manuel Valle Torre
- Foco: LLMs como ferramenta de intervenção em cursos de programação (CS education)
- Relacionado com a linha de pesquisa de Baker e Pankiewicz (UPenn) sobre feedback automático

## Relevância para este TCC
Background para a direção futura (TCC 2 — Deployment). Não relevante para a fase atual de modelagem e avaliação. Citar apenas se for discutir intervenções ou trabalhos futuros.
